# SalesTrainer — ROADMAP

## Как агент работает с этим файлом
1. Читай фазы сверху вниз
2. Найди первый блок со статусом `[ ]`
3. Декомпозируй его на подзадачи самостоятельно
4. Выполни → отметь `[x]` → коммит → следующий блок
5. Если блок заблокирован — отметь `[~]` и запиши причину рядом

## Статусы
- `[ ]` — не начато
- `[>]` — в процессе (агент работает прямо сейчас)
- `[x]` — готово
- `[~]` — заблокировано (причина рядом)
