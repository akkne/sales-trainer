# Tech Stack

## Backend
- **.NET 9** — ASP.NET Core Web API
- **Entity Framework Core** — ORM (PostgreSQL)
- **FluentValidation** — request validation
- **Serilog** — structured logging (Loki sink)
- **Hangfire** — background jobs (PostgreSQL storage)
- **Swashbuckle** — OpenAPI/Swagger

## Frontend
- **Next.js 16** — React framework
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **TanStack Query** — data fetching
- **Zustand** — state management
- **Framer Motion** — animations
- **Vitest** — testing

## Databases
- **PostgreSQL 17** — primary relational DB
- **MongoDB 8** — document storage
- **Redis 7** — caching

## Observability
- **Grafana** — dashboards
- **Loki** — log aggregation
- **Prometheus** — metrics

## Auth
- JWT (Bearer tokens)
- Google OAuth

## Infrastructure
- Docker Compose
